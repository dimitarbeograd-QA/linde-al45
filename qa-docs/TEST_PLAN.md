# Test Plan — Linde AL45 Интерактивна схема

## 1. Обхват
Уеб приложение с интерактивна SVG/растерова схема на части за мотокар Linde AL45.
Включва zoom/pan навигация, търсене и филтриране на части, лупа (magnifier),
попъп карти за части (инфо/редакция), панел със схеми, поръчки, скенер, табло
(dashboard), одит лог и управление на потребители (admin).

## 2. Цели на тестването
- Диаграмата да се зарежда коректно на desktop и мобилни устройства.
- Zoom/pan/drag взаимодействията да работят плавно и без визуални артефакти.
- Търсенето и филтрирането на части да връщат точни резултати.
- Ролево-базираните функции (admin панели: users, audit, dashboard) да са скрити/недостъпни за обикновени потребители.
- Поръчки, скенер и печат да произвеждат коректни данни/документи.

## 3. Тестова среда
| Компонент | Детайли |
|---|---|
| Браузъри | Chrome, Firefox, Edge, Safari (desktop + mobile) |
| Устройства | Desktop (мишка), Tablet/Mobile (touch, pinch-zoom) |
| Разделителни способности | 360px, 768px, 1024px, 1440px+ |
| Достъп | index.html (статичен файл / hosted) |

## 4. Видове тестове
1. **Функционално тестване** — zoom контроли, търсене, филтри, попъп карти, панели.
2. **UI/UX тестване** — responsive поведение, hamburger меню, overlay-и.
3. **Touch/gesture тестване** — pinch-to-zoom, drag за преместване на номера.
4. **Regression тестване** — след всяка промяна в диаграмата/схемите.
5. **Достъпност (a11y)** — контраст, focus states на бутоните в header менюто.
6. **Security/Roles** — проверка, че admin функции (users, audit) не са достъпни без логин.

## 5. Roles и достъп
| Роля | Достъп |
|---|---|
| Гост/Потребител | Zoom, търсене, преглед на части, поръчки |
| Admin/Собственик | Dashboard, Audit панел, Users панел, редакция на части |

## 6. Критерии за приемане
- Няма счупени zoom/pan контроли в нито един поддържан браузър.
- Търсенето връща 0 фалшиво-положителни/отрицателни резултата при точни термини.
- Admin функционалности не изтичат данни към неоторизирани потребители.
- Няма конзолни грешки (JS errors) при стандартен flow.
