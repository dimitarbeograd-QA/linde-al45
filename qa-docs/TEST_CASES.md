# Тест кейсове — Linde AL45 Интерактивна схема

Легенда: **P** = Priority (P1 критичен, P2 висок, P3 среден)

## Zoom & Pan (диаграма)

| ID | Стъпки | Очакван резултат | P |
|---|---|---|---|
| TC-01 | Отвори страницата, натисни бутон „+" (zoomIn) няколко пъти | Диаграмата се увеличава плавно, без preлив извън стейджа | P1 |
| TC-02 | Натисни бутон „−" (zoomOut) до минимум | Диаграмата се смалява, спира на min zoom без счупване на layout | P1 |
| TC-03 | Натисни „⊡" (zoomFit) след ръчен zoom | Диаграмата се центрира и мащабира да се побере в стейджа | P2 |
| TC-04 | Влачи диаграмата (drag) при увеличен zoom | Диаграмата се движи плавно в посоката на влаченето | P1 |
| TC-05 | На touch устройство — pinch to zoom | Zoom нивото се променя пропорционално на жеста | P1 |
| TC-06 | Натисни „◎" (btnDebug) | Показват се дебъг зоните/номерата на частите | P3 |
| TC-07 | Натисни „✥" (btnDragMode) и премести номер на част | Номерът се премества и позицията се запазва | P2 |
